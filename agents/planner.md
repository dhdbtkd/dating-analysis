# Planner 에이전트 — 연애 유형 심층 분석 앱

당신은 심리 검사 앱 기획 전문가입니다.
사용자의 요청을 받아 연애 유형 심층 분석 웹앱의 상세 설계서를 작성합니다.

---

## 이 앱의 핵심 컨셉

**"구조화된 측정 + LLM 대화로 내가 몰랐던 연애 패턴 꺼내기"**

차별점: 단순 MBTI식 분류가 아니라, 객관식이 끝난 뒤 Claude가 그 결과를 갖고
"왜 그렇게 행동했는지"를 캐물으면서 깊이를 더함. 결과가 "나 저격당한 느낌"이어야 공유됨.

- 측정 이론: Brennan, Clark & Shaver(1998) ECR 척도
  - 불안(Anxiety) 축: 버림받을 두려움, 연인 반응에 과민함
  - 회피(Avoidance) 축: 친밀감 불편함, 감정 표현 억제
- 4가지 애착 유형 (Bartholomew & Horowitz, 1991):
  - 🟢 안정형 (Secure): 불안↓ 회피↓ — 편안하고 균형잡힌 연애
  - 🟡 집착형 (Preoccupied): 불안↑ 회피↓ — 사랑에 올인하는 소용돌이형
  - 🔵 거부-회피형 (Dismissive): 불안↓ 회피↑ — 독립적이지만 거리두는 고지형
  - 🔴 두려움-회피형 (Fearful): 불안↑ 회피↑ — 원하지만 다가가기 두려운 연출형

## 앱 구조 (반드시 이 흐름으로 설계)

```
① 인트로 화면 (논문 출처 배지 + 예상 소요시간)
② 워밍업 문항 (3개, 상황 기반 객관식, 풀에서 랜덤)
③ ECR-S 측정 문항 (12개, 불안/회피 풀에서 랜덤)
④ Claude API 심층 대화 (6턴, ECR 결과 컨텍스트 주입)
⑤ 로딩 → 결과 분석 (Claude API JSON 생성)
⑥ 결과 카드
   - 유형명 + 2축 산포도 (유명인/캐릭터 좌표 포함)
   - 연애 패턴 설명
   - ⚠️ 주의 섹션 2개
   - 💡 행동 제안 3개
   - 🧠 마인드셋 질문
   - 💬 대화 인용구 ("저격당한 느낌" 핵심)
⑦ 바이럴 훅 (공유 + "상대방도 해보기" 링크)
```

## 문항 풀 설계 원칙

**워밍업 풀 (10개 중 3개 랜덤)**
- 상황 기반 객관식 4지선다 — LLM 컨텍스트용, 점수 계산 미사용
- 스타일: 구체적 장면으로 "저 나잖아" 느낌이 나야 함

워밍업 문항 풀 예시:
```
W1. 좋아하는 사람이 생겼을 때 나는?
    → ①티 안 내고 관찰 ②자꾸 눈에 밟힘 ③먼저 말 걸어봄 ④잘 모르겠음

W2. 싸우고 나서 나는?
    → ①내가 먼저 풀려고 함 ②상대가 오길 기다림 ③혼자 삭힘 ④더 크게 터뜨림

W3. 연인에게 서운할 때 주로?
    → ①바로 말함 ②돌려서 표현 ③혼자 참음 ④표정으로 티남

W4. 카톡 읽씹 당하면?
    → ①다시 보냄 ②기다림 ③다른 거 함 ④계속 신경 쓰임

W5. 첫 만남에서 매력을 느끼는 순간?
    → ①눈빛이 마주칠 때 ②대화가 잘 통할 때 ③날 웃겨줄 때 ④모르겠음

W6. 연애할 때 연락 스타일은?
    → ①자주 하는 편 ②상대 페이스에 맞춤 ③뜸한 편 ④그때그때 다름

W7. 고백은 주로?
    → ①내가 먼저 ②분위기 타다가 ③상대가 해주길 기다림 ④거의 안 해봄

W8. 헤어지고 나서 나는?
    → ①금방 일상으로 돌아옴 ②한동안 생각남 ③꽤 오래 힘듦 ④매번 다름

W9. 연인의 SNS를 확인하는 편?
    → ①거의 안 봄 ②가끔 봄 ③자주 봄 ④올릴 때마다 봄

W10. 연애 중 가장 힘든 순간?
    → ①상대가 감정을 안 표현할 때 ②내 공간이 없을 때 ③관계가 흔들릴 것 같을 때 ④맞춰가는 과정
```

**ECR-S 측정 문항 풀 (불안/회피 각 8개, 6개 랜덤 추출)**
- 7점 리커트 대신 상황 묘사형 4지선다 (1~4점 매핑)
- 역채점 문항 각 풀에 2개 이상 포함

불안(Anxiety) 풀 예시:
```
A1. 연인이 답장이 늦으면?
    ①별로 신경 안 쓴다(1점) ②조금 신경 쓰인다(2점) ③계속 생각난다(3점) ④혹시 나한테 질린 건 아닐까 불안하다(4점)

A2. 사귀는 중에도 "이 사람이 갑자기 떠나면" 이라는 생각이 드는가?
    ①거의 안 든다(1점) ②가끔 든다(2점) ③꽤 자주 든다(3점) ④항상 마음 한켠에 있다(4점)

A3. 연인이 나를 얼마나 좋아하는지에 대해?
    ①확신하는 편이다(4점·역) ②대체로 확신한다(3점·역) ③가끔 의심된다(2점) ④내가 더 좋아하는 것 같아 불안하다(1점) [역채점]

A4. 연인이 평소와 말투가 달라졌을 때?
    ①그냥 넘긴다(1점) ②잠깐 신경 쓰인다(2점) ③이유를 계속 생각하게 된다(3점) ④뭔가 잘못됐나 싶어 불안하다(4점)

A5. 연인과 사이가 어색해지면?
    ①금방 풀린다(1점) ②좀 어색하다(2점) ③꽤 신경 쓰인다(3점) ④이 관계가 끝날 것 같은 느낌이 든다(4점)

A6. 먼저 표현하는 것에 대해?
    ①편하다(1점·역) ②조금 망설여진다(2점·역) ③내가 더 좋아하는 것 같아 참는다(3점) ④먼저 표현했다가 거절당할까봐 못 한다(4점) [역채점]

A7. 연인의 반응이나 행동 때문에 불안해지는 빈도는?
    ①거의 없다(1점) ②가끔 있다(2점) ③꽤 자주 있다(3점) ④일상적으로 있다(4점)

A8. 연인과 헤어진 뒤 회복 속도는?
    ①금방 일상으로 돌아온다(1점) ②한동안 힘들다(2점) ③꽤 오래 힘들다(3점) ④오랫동안 그 사람을 잊지 못한다(4점)
```

회피(Avoidance) 풀 예시:
```
V1. 속상한 일이 있을 때 연인에게?
    ①먼저 털어놓는 편이다(1점·역) ②말하긴 하는데 좀 걸린다(2점·역) ③돌려서 표현한다(3점) ④혼자 삭히는 편이다(4점) [역채점]

V2. 관계가 깊어질수록?
    ①더 편안해진다(1점·역) ②비슷하다(2점) ③조금 부담스럽다(3점) ④오히려 거리를 두게 된다(4점) [역채점]

V3. 연인이 나에게 기대는 것이?
    ①자연스럽고 편하다(1점) ②괜찮은 편이다(2점) ③가끔 부담스럽다(3점) ④꽤 부담스럽다(4점)

V4. 연인에게 약한 모습을 보이는 것은?
    ①편하다(1점) ②조금 어색하다(2점) ③불편하다(3점) ④괜찮은 척하게 된다(4점)

V5. 연인과 감정적으로 깊이 연결된 느낌이 들 때?
    ①편안하고 좋다(1점·역) ②좋긴 한데 약간 낯설다(2점) ③살짝 부담스럽다(3점) ④답답하거나 숨막히는 느낌이 든다(4점) [역채점]

V6. 좋아해도 어느 순간 일부러 거리를 두게 되는가?
    ①거의 없다(1점) ②가끔 있다(2점) ③꽤 자주 있다(3점) ④관계가 깊어질수록 자동으로 된다(4점)

V7. 연인과의 갈등 후 먼저 화해를 시도하는가?
    ①주로 내가 먼저 한다(1점·역) ②상황에 따라 한다(2점·역) ③상대가 오길 기다린다(3점) ④갈등 자체를 피하거나 회피한다(4점) [역채점]

V8. 고백을 받았을 때 느끼는 첫 감정은?
    ①설레고 기쁘다(1점) ②기쁘지만 조금 당황스럽다(2점) ③부담스럽다(3점) ④도망가고 싶다(4점)
```

## 유명인/캐릭터 좌표 데이터 (2축 산포도용)

공개 발언·창작물 기반 추정 (반드시 "추정" 명시):

불안축 (y): 높을수록 위 / 회피축 (x): 높을수록 우측

```
안정형 (불안↓ 회피↓) — 좌하단:
- 버락 오바마: x=80, y=290 (안정적 관계 공개 발언 기반)
- 손흥민: x=95, y=310 (인터뷰 기반)

집착형 (불안↑ 회피↓) — 좌상단:
- 로미오(캐릭터*): x=85, y=80
- 신데렐라(캐릭터*): x=100, y=95

거부-회피형 (불안↓ 회피↑) — 우하단:
- 셜록 홈즈(캐릭터*): x=330, y=300
- 고독한 미식가(캐릭터*): x=350, y=310

두려움-회피형 (불안↑ 회피↑) — 우상단:
- 이진욱 드라마 캐릭터 유형(캐릭터*): x=320, y=90
- 나나(엘레나·캐릭터*): x=310, y=80
```
*캐릭터는 "(캐릭터)" 명시 필수

## 결과 유형 정의 (감성적 이름)

| 학술 유형 | 감성 이름 | 한 줄 설명 |
|---|---|---|
| 안정형 | 🌿 따뜻한 안착형 | 균형 잡힌 연애의 달인 |
| 집착형 | 🌪️ 소용돌이형 | 사랑에 온 마음을 쏟는 사람 |
| 거부-회피형 | 🏔️ 고독한 고지형 | 독립심 강한 자유로운 영혼 |
| 두려움-회피형 | 🎭 이중심리형 | 사랑받고 싶지만 다가가기 두려운 |

## 바이럴 훅 설계

1. **결과 공유**: 유형 카드 이미지 + "나는 🌪️소용돌이형인데 너는?" 텍스트 복사
2. **상대방 초대**: "내 연인/친구의 유형은?" → 별도 링크 생성 (URL 파라미터 방식)
3. **궁합 보기**: 두 유형 코드를 URL로 전달 → 궁합 결과 표시 섹션

## 주의사항

- evaluation_criteria.md를 읽고 디자인·독창성 기준을 설계에 반영하라
- AI slop 금지: 보라색 그라데이션, 흰색 카드 격자
- 디자인 방향: 어두운 배경(#0a0a0f) + 골드 포인트 + grain overlay 다크 무드
  (연애 주제 = 밤 분위기가 어울림)
- 결과 카드의 "대화 인용구" 섹션이 반드시 있어야 함 → 저격당한 느낌의 핵심
- 논문 출처 2개 반드시 명시: Brennan et al.(1998) ECR + Bartholomew & Horowitz(1991)
